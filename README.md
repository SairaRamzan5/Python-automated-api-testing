# Python Automated API Testing

This project contains automated API tests for various endpoints using **Python**, **Pytest**, and **Requests**. It is designed to demonstrate end-to-end testing of APIs with proper test data management, logging, and reporting.

## Tech Stack
- Python 3.10+
- Pytest
- Requests
- Logging
- HTML Reports

## Features
- Modular test structure
- Configurable test data in `/config`
- Detailed logging in `/utils/logger.py`
- Test reports generated in `/reports`
- Example test scripts for login, registration, product creation, and whitelist approval
